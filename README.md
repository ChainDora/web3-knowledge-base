# Web3 全栈知识库 (Web3 Knowledge Base CN)

## 我们是谁？
我们是JRBC区块链协会，目标是建设成中文区高校较有影响力的区块链协会

## 我们在做什么？
这是一个开源、中文优先的 Web3 全栈知识库项目。目标是把 Web3 世界真正重要的知识（密码学、共识、安全、DeFi、L2、RWA、AI×Crypto、治理和市场机制）结合真正的市场系统化、成体系同时尽可能结合底层代码写清楚，而不是零散的营销内容。

我们希望它成为：
- 新开发者 / 学生的系统教材
- 研究员、投资人、合规相关角色的快速索引库
- 中文社区长期可复用的公共基础设施

本项目将持续开放协作，所有内容都会以开源协议发布，允许引用、二次创作和教学使用。

## 为什么需要这个项目？
- 中文世界的区块链/Web3教程大多是投机导向而不是工程导向
- 新人很难从“原理”走到“真实可部署的代码”
- 高校/黑客松/投研团队/DAO运营需要可复用的系统化资料
- 关键主题（L2 扩容、RWA、跨链桥安全、DAO 治理攻击）几乎没有成体系同时结合真实市场的中文教材

我们想把这些缺口补上，并且永远开源。
