# 附录：资源导航、术语表、案例库

这一部分做三件事：
1. 把你在学习/研究/开发过程中最常用的公共资源都集中列出来，形成“起点导航”。
2. 整理术语速查表（中英对照），方便新人不被英文攻击性名词吓跑。
3. 做“实战案例”索引，比如白皮书、黄皮书、关键协议的源码仓库，以及安全事件复盘。

这些内容本质上是一个“工具箱”，而且会长期更新。
